# 請讀我

這是 Sunshine + Moonlight 用的虛擬螢幕安裝工具。

## 安裝

1. 下載 release 裡的 `SunshineVddHelper.zip`。
2. 解壓縮。
3. 右鍵 `Install.ps1`。
4. 選「使用 PowerShell 執行」。
5. 跳出系統管理員權限時按「是」。
6. 安裝完成後重開 Sunshine，必要時重開機一次。

安裝後 Moonlight 串流會使用虛擬螢幕。串流結束後會自動還原原本的實體螢幕布局。

## 全螢幕沒有黑邊

要沒有黑邊，Moonlight 選的解析度比例要接近裝置螢幕比例。

已內建常用解析度：

- `1920x1080`：一般 16:9 螢幕
- `640x480`：4:3 小螢幕
- `2556x1179`：iPhone 15 Pro 橫向比例
- `2400x1080`：20:9 手機相容模式
- `3200x1440`：Poco F5 Pro 橫向比例

Moonlight 裡請把解析度選成對應裝置的比例。比例不對時，只能在黑邊、裁切、拉伸變形之間選一種。

## 解除安裝

1. 右鍵 `Uninstall.ps1`。
2. 選「使用 PowerShell 執行」。
3. 跳出系統管理員權限時按「是」。

解除安裝會移除 Sunshine 設定與開機排程，但不會自動移除虛擬顯示器驅動。若要完全移除，請到 Windows 裝置管理員手動移除 `Virtual Display Driver`。

## 注意

- 需要 Windows 10 或 Windows 11。
- 需要已安裝 Sunshine。
- 安裝時需要網路，會下載 Virtual Display Driver 和 MultiMonitorTool。
- 支援多實體螢幕，會保存並還原整個螢幕布局。
- 不綁定任何固定電腦、螢幕編號或 Sunshine device id。
- Sunshine 會嘗試依 Moonlight 要求自動切換虛擬螢幕解析度。
